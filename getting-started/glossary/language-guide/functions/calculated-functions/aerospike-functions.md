# Aerospike functions

## `AEROSPIKE_KV`

Gets or sets the value in Aerospike.

#### Inputs

* **keys** - Keys to set
* **values** - Values to set / get if the keys already exists

#### Properties

* **Connection**
* **Batch Size**
* **Namespace**
* **Set Name**
* **Bin**
* **Reverse Set Name**
* **Reverse Bin**
